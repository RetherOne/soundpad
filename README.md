# SoundBoard — Virtual Microphone Soundboard

Soundboard с горячими клавишами: подмешивает звуки к вашему микрофону в реальном времени.
Собеседник в Discord/Zoom/TeamSpeak слышит и вас, и звуки одновременно.

---

## Установка (один раз)

### 1. VB-Cable (виртуальный аудио кабель)
Скачайте и установите бесплатно:
👉 https://vb-audio.com/Cable/

После установки **перезагрузите компьютер**.

### 2. Python и зависимости

```bash
pip install -r requirements.txt
```

> Python 3.8+ обязателен. Скачать: https://python.org

---

## Запуск

```bash
python soundboard.py
```

## Настройка

### Шаг 1 — Устройства (кнопка ⚙ УСТРОЙСТВА)

| Поле | Что выбрать |
|------|-------------|
| 🎤 Микрофон | Ваш реальный микрофон |
| 🔌 VB-Cable Input | `CABLE Input (VB-Audio Virtual Cable)` |
| 🎧 Мониторинг | Ваши наушники/колонки |

### Шаг 2 — Discord / Zoom
В настройках голосового приложения выберите микрофон:
`CABLE Output (VB-Audio Virtual Cable)`

> Важно: выбирайте **CABLE Output**, а не Input.
> — Input это "вход" куда пишет ваше приложение
> — Output это "выход" откуда читает Discord

### Шаг 3 — Добавьте звуки
Нажмите **✎** на любую кнопку или ПКМ → откроется редактор слота.

## Использование

- **▶ на кнопке** — воспроизвести звук через виртуальный микрофон
- **Горячая клавиша** — нажмите назначенную клавишу (работает если окно в фокусе)
- **⏹ СТОП SFX** — экстренная остановка всех звуков
- **МУТ MIC** — заглушить микрофон (только звуки)
- **Ползунки** — раздельная громкость микрофона и звуков

## Часто задаваемые вопросы

**Q: Собеседник не слышит звуки**
A: Проверьте что в Discord выбран `CABLE Output`, а не ваш обычный микрофон.

**Q: Звуки слышно с задержкой**
A: Попробуйте уменьшить `BLOCKSIZE` с 1024 до 512 в коде (`AudioEngine.BLOCKSIZE`).

**Q: Качество звука плохое**
A: В Discord отключите шумоподавление: Settings → Voice → Advanced → отключите всё.

**Q: VB-Cable не найден в списке**
A: Убедитесь что установили и перезагрузили компьютер после установки.
